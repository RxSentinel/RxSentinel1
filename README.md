# 💊 RxSentinel

## 🧠 Overview
RxSentinel is a smart drug interaction detection system designed to identify potentially harmful combinations of medicines and improve patient safety.

It allows users to input multiple medicines or upload prescription images and provides instant analysis with risk levels and explanations.

---

## 🚨 Problem Statement
Patients often take multiple medications without knowing their interactions, which can lead to serious side effects or life-threatening conditions.

Existing solutions are:
- Complex to use
- Not accessible to common users
- Require medical knowledge

---

## 💡 Solution
RxSentinel provides a simple and user-friendly platform where users can:
- Enter multiple medicines
- Upload prescription images
- Get instant interaction analysis
- View risk levels and reasons

---

## ⚙️ How It Works
1. User inputs medicines or uploads an image  
2. System processes and normalizes drug names  
3. Matches combinations with a structured dataset  
4. Displays:
   - Risk level (High / Medium / Low)
   - Reason for interaction
   - Safety warning
